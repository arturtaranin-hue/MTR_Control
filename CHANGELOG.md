# Changelog

All notable changes to MTR_Control will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses semantic versioning.

## [0.1.0] - 2026-07-23

### Added

- Created foundational VBA project structure.
- Added core modules for application entry point, logging, settings, utilities, and error-handling conventions.
- Added domain model skeletons for materials, movements, and arrivals.
- Added workbook repository and import service skeletons.
- Added dashboard form placeholder with no business logic.
- Added architecture, data dictionary, and algorithm documentation.
